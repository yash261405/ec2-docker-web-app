# 🚀 EC2 Docker Web App

## 📌 Project Overview
Deployed a containerized web application on AWS EC2 using Docker and exposed it to the internet.

## 🏗️ Architecture
User → EC2 → Docker Container → Nginx → Web App

## ⚙️ Tech Stack
- AWS EC2
- Docker
- Nginx
- Linux

## 🚀 Deployment Steps
1. Launched EC2 instance (Amazon Linux)
2. Installed Docker
3. Created custom HTML app
4. Built Docker image using Dockerfile
5. Ran container with port mapping (80:80)
6. Accessed via public IP

## 🌍 Live Output
Application is accessible via EC2 Public IP

## 🧠 Key Learnings
- Cloud deployment on AWS
- Containerization using Docker
- Debugging real-world issues (ports, services)
- Networking and security groups
